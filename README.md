Workout & Nutrition Tracker
A simple Streamlit app for logging workouts and nutrition. Customized for back-safe routines.
Deployment

Clone this repo.
Install dependencies: pip install -r requirements.txt
Run locally: streamlit run app.py
Deploy to Streamlit Sharing: Connect your GitHub repo at streamlit.io.

Usage

Log entries daily.
Export CSVs for analysis.
Public for anyone to fork/use.
